## What's Included

* 📊 Ready-made timer resolution presets and .bat / .cmd scripts (0.5ms–1ms targets)
* 📈 Performance measurement templates & benchmark notes
* 🔍 System check checklists (before/after latency tests)
* 🗓️ Recommended application orders & startup presets
* 📸 Screenshot examples, graphs, and usage notes

## How to use

1. Download the archive from the **Releases** tab on the right → select the desired .zip file
2. Extract it to any folder
3. Run the included setup scripts or apply configurations manually
4. For presets: execute .bat/.cmd files as administrator → monitor current timer resolution
5. Test in-game or with latency tools → adjust values based on your hardware
6. Customize for your setup (Windows 10/11, specific games like CS2/Valorant/Fortnite) and re-test
